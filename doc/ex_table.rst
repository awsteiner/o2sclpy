Table example
=============

:ref:`O2sclpy <o2sclpy>`

This example loads a simple O2scl table and makes a simple plot of the
data. It uses the ``cap_cout()`` function to capture ``std::cout``
output from the table ``summary()`` functions.
     
Use this `link
<https://nbviewer.org/urls/raw.githubusercontent.com/awsteiner/o2sclpy/main/examples/table.ipynb>`_
to view this example as a jupyter notebook on nbviewer.org.
